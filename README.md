# 🎓 DIU OpenGL Campus Simulation

A **Computer Graphics project built with C++, OpenGL, and GLUT** that simulates a dynamic road scene in front of **Daffodil International University (DIU)**.
The project demonstrates real-time animation, environment effects, and interactive controls.

The scene includes moving buses, walking students, weather modes (day, night, rain), clouds, birds, thunder effects, and animated campus surroundings.

---

## 📸 Project Overview

This project recreates a lively university road environment using **OpenGL 2D graphics**.
Different animated elements simulate daily campus activity.

Key elements in the scene include:

* DIU entrance gate
* Moving DIU buses
* Students entering and exiting campus
* Animated clouds and birds
* Trees and landscape
* Day, night, and rain weather modes
* Thunder and lightning effects
* Bus stopping and student drop-off animation

---

## 🎮 Keyboard Controls

| Key   | Action                     |
| ----- | -------------------------- |
| **D** | Switch to Day Mode         |
| **N** | Switch to Night Mode       |
| **R** | Switch to Rain Mode        |
| **E** | Stop bus and drop students |
| **S** | Resume bus movement        |

## ▶ How to Run the Project

### 1️⃣ Install OpenGL and GLUT

Make sure you have:

* **g++ compiler**
* **OpenGL**
* **FreeGLUT / GLUT library**

---

### 2️⃣ Compile the Code

Example (Windows / CodeBlocks / Dev-C++):

```
g++ main.cpp -lglut -lglu -lopengl32
```

---

### 3️⃣ Run the Program

After compiling, run the generated executable file.

The OpenGL window will open and show the animated DIU campus scene.
